About World Planner

World Planner lets you design Growtopia worlds on a canvas using the complete item database. Blocks autotile automatically based on their neighbours, you can paint transparent colour overlays using real in-game paint items, and use the wrench tool to manually pick any texture variant from a block's spritesheet.

Tools & Features

Pencil, line, rectangle, circle and flood-fill drawing · Select / cut / copy / paste regions · Paint brush with in-game paint colours · Wrench texture override · Undo / redo (60 steps) · Grid and coordinate toggles · World presets · Export and import as JSON · Render & download as PNG
